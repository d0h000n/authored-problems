# authored-problems
Algorithm problems I've authored, packaged with Polygon for use across online judges.

## About

This repository hosts a collection of competitive programming problems authored by **Dohoon Kim (김도훈)**. Each problem is provided as a [Polygon](https://polygon.codeforces.com/) package, which includes statements, test data, validators, checkers, and reference solutions — ready to be imported into Polygon-compatible online judges (Codeforces Gym, QOJ, Universal Cup, DOMjudge, etc.).

## Problem List

- 🟢 **Warm-up** — Introductory problems for beginners
- 🔵 **Easy** — Straightforward problems requiring a single key idea
- 🟡 **Medium** — Problems requiring solid algorithmic background
- 🔴 **Hard** — Challenging problems requiring deep thought and synthesis

## Usage Policy

These problems are made publicly available for educational use, practice, and reuse on online judges, **subject to the following conditions**:

1. **Notify the author before reuse.** If you intend to host any of these problems on an online judge or use them in a contest, please reach out to me beforehand. This is so I can keep the list above up to date and coordinate with other organizers.
2. **Credit the author and original contest.** Any reuse must clearly attribute the problem to **Dohoon Kim (김도훈)** and cite the original contest listed in the table above.
3. **No modifications.** The problems must be used as-is. Translation into other languages is permitted, but altering the problem statement, constraints, test data, or intended solution is not allowed.

For any inquiries, please open an issue in this repository or contact me directly.
